🎯 Sobre Este Repositório

Este website estático funciona como a página central (Versão 1.0) do meu repositório de portfólio no GitHub. O objetivo principal deste espaço é organizar de forma profissional todos os projetos desenvolvidos ao longo da minha trajetória académica e pessoal.

Através do versionamento rigoroso com Git e da integração com plataformas externas, este repositório demonstra competências em engenharia de software, organização de arquitetura de diretórios e documentação limpa.

🛠️ Tecnologias e Metodologia

O fluxo de trabalho cumpre as melhores práticas de mercado recomendadas para o desenvolvimento de software:

  • Git Local: Utilização dos três estados do Git (Working Directory, Staging Area e Local Repository) com commits claros e semânticos.
  • GitHub Remoto: Repositório público configurado para servir de portfólio focado na atração de recrutadores e networking técnico.
  • GitHub Pages: Hospedagem e publicação automatizada deste website estático a partir do branch principal.
