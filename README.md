## 💻 Projeto em Destaque

Consiste em uma plataforma web estática de alta performance, com design responsivo em Dark Mode, totalmente versionada com Git e hospedada em nuvem.

* 🌐 **Acesse a página publicada em produção aqui:** [Visualizar PortfolioHUB Ativo](https://claraana309.github.io/github-page/)

### 🛠️ Tecnologias & Engenharia Aplicada
* **HTML5 Semântico:** Estruturação limpa e acessível.
* **CSS3 Moderno:** Estilização com variáveis globais (`:root`) e total responsividade (Mobile Friendly).
* **Git & GitHub:** Governança de código e histórico de commits semânticos.
* **Google GEMINI:** IA assistiva utilizada na versão Flash 3.5, para o planejamento arquitetural e validação de conformidade técnica do ecossistema.
